# 功能模块全景（含每个模块的来由）

按当前导航（App.tsx 6 个顶层页）整理，括号内是用户当初提它的原因/原话要点。

## 🧠 脑库 BrainPage
- 交易纪律与买卖心法库。来由：用户想要「一直进化、独属于我的交易 agent」，看到好复盘就发给它、自动提取逻辑。
- 支持**截图识别**导入（看到好文章直接截图发）。每天可**自动导入**论坛/文章内容（优先中国来源，越多越好）。
- 演变：一度被改成"规则库"，后又**作为独立页放回顶层**（2026-06-16，因为旧规则在新规则库里改不了）。

## 🎯 规则库 RuleLibraryPage（选股规则）
- 由脑库演化（2026-06-10）：系统按规则自动筛股，规则可命名/收藏，点开看筛出的股票。
- **两区分离**：上=我自己建的规则，下=系统**每日推送**的规则（自动从新闻提取构成规则→选股，不好用就删）。
- 规则支持：数值/形态条件、题材(行业成分)、**截图识别建规则**(多张拖入批量解析)、**回测**(与大盘对比)。
- 形态条件需多给几个可选项，支持自定义。筛出的股票要**预缓存**详情，点开即看（行业/主营/近期表现，砍掉总市值等没用的列）。

## 🏢 AI办公室 OfficePage
- 8 位 AI 专家，可咨询/讨论后给解答。来由：用户下载了 **TradingAgents-CN**，想把里面多个 agent 嵌进来设个"办公室"。
- 要求：**用角色身份不用人名**；每个 agent 能**调用工具**(数据接口/新闻/技术指标)，不是纯对话；要有**对话记忆**(同一对话框可持续对话)。模型 deepseek-v4-flash。

## 🏠 今日（today）
- **持仓 PortfolioPage**：截图识别批量录入(微信多张)、实时更新(开盘时,收盘/周末不更新)、今日操作记录(买卖/数量/价格/理由→AI分析+操作得分)、**卖出指导**(会买是徒弟会卖是师傅,走势+量能主导)。
- **我的自选 / 今日推荐 / 明日预判 / 推荐历史**：原是自选页内 4 个 tab，2026-06-10 提到与持仓并列。
  - 今日推荐：盘中实时更新可入股票；明日预判：休盘后给。推荐要**结合热点+消息+资金+技术面**挑能持仓的，不是看涨停堆。推荐历史可回看推荐逻辑(曾抱怨买了中远海控找不到当初推荐理由)。

## 📊 市场（market）
- **大盘行情 MarketPage**：首页直接铺行业框，点开看行业内股票，**龙头排最前**(像东财/同花顺)。
- **市场复盘 MarketReviewPage**：多维度复盘+信息层面总结，每天记录做**日历**可调日期(2026-06-10)。
- **涨停板 LimitUpReviewPage**：仿同花顺"涨停板复盘"账号，每个涨停给逻辑分析，每天4点半前跑出、留存、日历回看(方案b 数据接口)。
- **龙虎榜 LhbPage**。

## 🔬 研究（research）
- **个股复盘 ReviewPage**：多角度(事件/财务/新闻/技术)。用户对它**反复不满意**、多次要求重做(最近一次 2026-06-15)——是长期痛点，重消息面/事件驱动。
- **行业选股 ScreenerPage**。
- **新闻分析 NewsImpactPage**：粘贴新闻链接识别 + 新闻推送(附相关股票影响)；从国际新闻反推A股；**热搜榜**(国内/国外各一，如马斯克IPO对国内影响)；国际新闻翻译成中文摘要。
